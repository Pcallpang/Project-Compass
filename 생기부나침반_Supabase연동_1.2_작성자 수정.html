주요 기능을 빠르게 안내해 드릴게요.
1
행사 일정 탭에서 활동 추가
상단 행사 일정 탭 → 활동 추가하기 버튼으로 새 행사를 등록하세요.
2
학생 관리로 참여 학생 등록
활동 카드의 학생 관리 버튼을 클릭해 참여한 학생들을 추가하세요.
3
특기사항 작성
학생을 선택하고 특기사항을 작성하세요. 2초 뒤 자동저장되며 1500byte 제한이 있어요.
4
공개 버튼으로 담임교사에게 전달
작성 완료 후 공개 버튼을 누르면 담임교사 화면에 표시돼요.
1
담임교사 탭 클릭
상단 담임교사 탭을 클릭하면 우리 반 학생 명단이 나타나요.
2
학생 선택
왼쪽 명단에서 학생을 클릭하면 해당 학생의 활동 특기사항이 오른쪽에 펼쳐져요.
3
특기사항 카드 체크로 합산
담당교사가 공개한 특기사항 카드를 체크하면 위쪽 합산 박스에 자동으로 합쳐져요.
4
복사 후 NEIS에 입력
합산 내용을 복사 버튼으로 복사한 뒤 NEIS 생기부에 붙여넣으세요.
🧭 생기부 나침반 도움말
기본 워크플로우
1
활동 추가
행사 일정 탭 → 활동 추가하기 버튼으로 행사 이름, 카테고리, 학기를 입력해 등록해요.
2
참여 학생 등록
활동 카드의 학생 관리 버튼 클릭 → 학생 이름·학년·반·번호를 입력하거나 엑셀로 일괄 등록해요.
3
특기사항 작성
학생 목록에서 이름 클릭 → 오른쪽 작성칸에 입력해요. 2초 뒤 자동저장되며 상단에 저장 시각이 표시돼요.
4
공개
작성 완료 후 공개 버튼을 눌러야 담임교사 화면에 보여요. 전체 공개 버튼으로 일괄 처리도 가능해요.
자주 묻는 질문
특기사항 글자 수 제한이 있나요?
1500byte 제한이 있어요. 한글 1자 = 2byte 기준이며, 초과 시 빨간색으로 경고가 표시돼요.
활동 카드의 진행률은 무엇인가요?
카드 하단의 바가 공개된 학생 수 / 전체 참여 학생 수를 나타내요. 노란색이면 미완성, 초록색이면 전원 공개 완료예요.
새로고침해도 로그인이 유지되나요?
네, 브라우저에 세션이 저장돼서 새로고침해도 자동으로 로그인 상태가 유지돼요.
기본 워크플로우
1
담임교사 탭 접속
상단 네비게이션의 담임교사 탭을 클릭해요. 관리자가 권한을 부여해야 접근할 수 있어요.
2
학생 선택
왼쪽 명단에서 학생 이름을 클릭하면 해당 학생이 참여한 활동의 특기사항이 오른쪽에 표시돼요.
3
특기사항 합산
특기사항 카드 왼쪽의 체크박스를 클릭하면 위쪽 합산 박스에 자동으로 합쳐져요. 드래그로 순서 조정도 가능해요.
4
복사 후 NEIS 입력
합산 내용을 직접 수정한 뒤 복사 버튼으로 클립보드에 복사해 NEIS 생기부에 붙여넣으세요.
자주 묻는 질문
담임교사 탭이 보이지 않아요.
관리자에게 담임교사 권한 부여를 요청하세요. 권한이 있어야 탭에 접근할 수 있어요.
특기사항이 보이지 않아요.
담당교사가 공개 버튼을 눌러야 담임교사 화면에 나타나요. 담당교사에게 공개 여부를 확인해 주세요.
합산 내용을 저장할 수 있나요?
저장 버튼을 눌러 임시 저장해 두고, 나중에 다시 열면 저장된 내용이 복원돼요.
